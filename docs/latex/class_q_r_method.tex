Perform an iteration of the QR method. 

This function performs an iteration of the QR method. First, the QR decomposition of the current matrix is computed. Then, the matrix is updated to be the product of the R and Q matrices. This process is supposed to converge to a triangular matrix with the eigenvalues on the diagonal.
