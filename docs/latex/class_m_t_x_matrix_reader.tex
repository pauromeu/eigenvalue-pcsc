\doxysubsection{Detailed Description}
Reader class for parsing matrices from MTX (Matrix Market) files. 

This class extends \doxylink{class_matrix_reader}{Matrix\+Reader} and provides an implementation for reading matrices stored in MTX format. Currently, only symmetric matrices are supported.
